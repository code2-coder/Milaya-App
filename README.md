# Milaya Clothing Brand 🌟

Welcome to the **Milaya Clothing** application repository! This is a premium, modern, and highly interactive e-commerce platform built to provide an exceptional shopping experience. Milaya specializes in crafting timeless clothing that beautifully blends tradition, elegance, and exceptional craftsmanship.

## 🚀 Features

- **Sleek & Premium UI:** A meticulously designed dark-themed header and a high-end cinematic loader give the application a luxurious feel.
- **Advanced Visual & Voice Search:** Features integrated voice and picture search directly from the search bar, allowing users to find clothing using AI-powered visual matching or microphone input.
- **Dynamic Collections Menu:** A fully animated, categorised mega-menu for effortless browsing of curated excellence.
- **Real-time Search:** Lightning-fast search suggestions as you type with search history tracking.
- **User Authentication:** Secure email and Google OAuth integration for seamless user accounts.
- **Responsive Design:** A fully responsive layout optimized for all devices, ensuring the shopping experience is flawless on mobile, tablet, and desktop.
- **Admin Dashboard:** A comprehensive dashboard for managing products, variants, orders, and customer reviews.

## 💻 Tech Stack

- **Frontend:** React, Tailwind CSS, Framer Motion (for elegant animations), Lucide React (for icons)
- **Backend:** Node.js, Express.js
- **Database:** MongoDB
- **AI/ML Integration:** TensorFlow.js (for visual product matching)

## 🛠️ Getting Started

### Prerequisites

- Node.js (v18+)
- MongoDB (Running locally or via MongoDB Atlas)

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/CodeFusionProjects/Milaya-App.git
   cd Milaya-App
   ```

2. **Install Frontend Dependencies:**
   ```bash
   cd frontend
   npm install
   ```

3. **Install Backend Dependencies:**
   ```bash
   cd backend
   npm install
   ```

4. **Environment Variables:**
   Create a `.env` file in the `backend` directory and add the necessary environment variables (e.g., MongoDB URI, JWT Secrets, API keys).
   Create a `.env` file in the `frontend` directory for any frontend-specific environment variables (e.g., `VITE_API_URL`).

### Running the Application

1. **Start the Backend Server:**
   ```bash
   cd backend
   npm run dev
   ```

2. **Start the Frontend Development Server:**
   ```bash
   cd frontend
   npm run dev
   ```

3. Open your browser and navigate to `http://localhost:5173` (or the port specified by Vite).
